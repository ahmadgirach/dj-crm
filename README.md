# dj-crm
A CRM app built with Django...TailwindCSS templates used in this project are from [Tailblocks](https://tailblocks.cc).

### Topics Learnt
- CRUD APIs
- Templates
- Signals
- Custom Mixins
- Forms / Django Crispy Forms
- Send Email
- Overriding existing auth APIs
